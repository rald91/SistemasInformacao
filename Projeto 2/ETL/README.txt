README

1. Este trabalho é fornecido sob a forma de um Jupyter Notebook (ETL.ipynb), onde é realizado o processo de ETL e a preparação dos dados utilizados na análise com técnicas de Machine Learning.

2. Ficheiros incluídos:

 - ETL.ipynb – Notebook principal com o código de pré-processamento e transformação dos dados

 - ETL.html – Versão do notebook em HTML, apenas para visualização

 - ine_habitacao.xlsx – Dataset original do INE

 - ine_habitacao_limpo.xlsx – Dataset resultante após o processo de ETL

3. Execução do notebook - Para abrir e executar o ficheiro ETL.ipynb é necessário:

 - Python 3.x

 - Jupyter Notebook ou Jupyter Lab

 - Biblioteca pandas

O notebook pode ser aberto executando, na linha de comandos, jupyter lab ou jupyter notebook.

4. Visualização sem Jupyter

Caso não esteja instalado o Jupyter, o conteúdo do trabalho pode ser consultado abrindo o ficheiro ETL.html em qualquer browser, sem necessidade de instalar software adicional.
